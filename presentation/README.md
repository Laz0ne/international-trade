# 📊 CMO Deck — Ferrari & Bori Supply Chain Strategy

Générateur automatique de présentation PowerPoint pour la stratégie supply chain.

## 🚀 Utilisation

```bash
cd presentation
npm install
npm run build
```

Cela génère le fichier **`CMO_Deck_Ferrari_Bori_FINAL.pptx`** dans le dossier `presentation/`.

## 📁 Contenu de la présentation (5 slides)

| Slide | Thème |
|-------|-------|
| 1 | Executive Summary — 3 KPI (100% dispo, 0 rupture, 2× air) |
| 2 | Le problème résolu — ruptures, containers, suivi |
| 3 | La solution en 3 piliers — océan, air, inventaire |
| 4 | Impact business — revenus, clients, vitesse, coûts |
| 5 | Recommandation finale avec KPIs marketing |

## 🛠️ Prérequis

- [Node.js](https://nodejs.org) v16+
- npm (inclus avec Node.js)

## 📤 Ouvrir la présentation

Le fichier `.pptx` généré est compatible avec :
- Microsoft PowerPoint
- Google Slides (importer le fichier)
- LibreOffice Impress
